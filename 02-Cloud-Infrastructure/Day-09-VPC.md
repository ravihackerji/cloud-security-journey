# Day 09 - AWS VPC & Azure VNet

## Topics Covered

- VPC
- VNet
- Public Subnet
- Private Subnet
- Internet Gateway
- Route Table
- Security Group
- Network ACL

---

## Key Learnings

- A VPC is a private cloud network.
- Azure VNet is the Azure equivalent.
- Databases should be placed in private subnets.
- Security Groups protect resources.
- Network ACLs protect subnets.

---

## Cloud Security Perspective

- Never expose databases directly to the internet.
- Use least privilege for network access.
- Audit Security Group rules regularly.
