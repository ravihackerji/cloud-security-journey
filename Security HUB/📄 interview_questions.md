# AWS Security Hub Interview Questions

## Beginner Level

### 1. What is AWS Security Hub?

Answer:

AWS Security Hub is a centralized security posture management service that aggregates security findings from AWS services and supported third-party tools.

---

### 2. What is the primary purpose of Security Hub?

Answer:

To provide a single dashboard for monitoring, prioritizing, and managing security findings across AWS environments.

---

### 3. Does Security Hub perform threat detection?

Answer:

No. Services such as Amazon GuardDuty perform threat detection. Security Hub aggregates and manages the resulting findings.

---

### 4. What is a Security Finding?

Answer:

A Security Finding is a security issue or recommendation generated by AWS services or integrated security tools.

---

### 5. Which AWS services commonly integrate with Security Hub?

Answer:

- Amazon GuardDuty
- AWS Config
- Amazon Inspector
- IAM Access Analyzer
- Amazon Macie
- AWS Firewall Manager
- AWS Audit Manager

---

## Intermediate Level

### 6. What is AWS Foundational Security Best Practices (FSBP)?

Answer:

It is a built-in AWS security standard containing recommended security controls to improve the security posture of AWS environments.

---

### 7. What is the CIS AWS Foundations Benchmark?

Answer:

It is an industry-recognized security benchmark that evaluates AWS environments against security best practices published by the Center for Internet Security.

---

### 8. What is the Security Score?

Answer:

The Security Score measures how well AWS resources comply with enabled security standards.

---

### 9. Can Security Hub automate incident response?

Answer:

Not by itself. However, it can integrate with Amazon EventBridge to trigger AWS Lambda, Amazon SNS, or AWS Systems Manager Automation workflows.

---

### 10. Why is Security Hub important for SOC teams?

Answer:

It centralizes findings from multiple security tools, helping analysts prioritize risks and investigate incidents more efficiently.

---

## Advanced Level

### 11. A company wants one dashboard for GuardDuty, Inspector, AWS Config, and Macie findings. Which AWS service should they use?

Answer:

AWS Security Hub.

---

### 12. Does Security Hub store CloudTrail logs?

Answer:

No. CloudTrail stores API activity logs. Security Hub references security findings generated by integrated services.

---

### 13. How does Security Hub improve compliance?

Answer:

It continuously evaluates resources against enabled security standards, provides compliance reports, and highlights failed controls.

---

### 14. How can Security Hub be integrated into automated remediation?

Answer:

By using EventBridge rules that trigger Lambda functions, SNS notifications, or Systems Manager Automation when new findings are generated.

---

### 15. How can Security Hub be automated using Python?

Answer:

Using Boto3 to:

- Retrieve findings
- Filter findings by severity
- Generate compliance reports
- Export findings to CSV
- Build security dashboards
- Integrate with ticketing or alerting systems
