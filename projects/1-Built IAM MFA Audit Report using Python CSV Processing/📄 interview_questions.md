# Interview Questions

## 1. Why is MFA important?

MFA provides an additional layer of authentication beyond passwords, reducing the risk of unauthorized access.

---

## 2. Why is CSV commonly used?

CSV is lightweight, easy to process, and widely supported by spreadsheet applications.

---

## 3. Why use csv.DictReader()?

It allows access to values using column names instead of indexes, making the code easier to read and maintain.
